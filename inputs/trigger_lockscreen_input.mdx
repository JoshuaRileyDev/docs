title: "Trigger Lockscreen Input"
description: ""
---

Lockscreen Input allows you to covertly input either a 4 digit or 6 digit number using a fake lockscreen.

You can then access the input via the Return Input shortcut and use shortcuts to process the input however you like.

There is optional initial configuration that you can do which is to set the wallpaper that the fake lockscreen will use as its background. You can set this by going to the Settings -> Lockscreen input.

To add this action into your routines you’ll need to search for Trigger lockscreen input via the iOS Shortcuts.

After you’ve added the action you’ll be able to set if you’d like to enter 4 digits or 6 digits in the fake lockscreen.

To do this press on the little arrow in the action, this will drop down the setting you’ll need to set to choose between 4 or 6 digits.
