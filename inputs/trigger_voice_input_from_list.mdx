---
title: "Trigger Voice Input From List"
---

This will likely be the most used way for this input method. This allows you to use the inbuilt shortcut action to create a list of key words that the voice recognition will detect and return back using the utility shortcut.

To get started you’ll first need to add the inbuilt List action.

You can then set a list of the different words you want the Voice input to recognise, if it recognises one of these words then it’ll set just that word as the input.

An important thing to note is that you need to make sure key words don’t include other key words, for example you can’t have Ace & also Ace of Hearts because it’ll first detect Ace and stop the detection.
