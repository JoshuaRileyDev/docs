---
title: "Trigger Keyboard Input"
---

This input method allows you to input any alphanumerical value by using a Bluetooth remote.

To set this up you’ll need to add the Trigger Keyboard Input action to your shortcut.

After you’ve inputted your first number or letter you’ll have 3 seconds to enter in subsequent values before it gets locked in and confirmed.

After you’ve added this action you can then add the Run Shortcut action and set it to be the WMP-Return Input shortcut.

Then you can use the shortcut output result to either set to a variable within shortcuts or use it in any of the other actions.
